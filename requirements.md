Functional Requirements

Upload oral images or medical scans via mobile/web app
Pre-process images (noise removal, normalization, ROI extraction)
Run AI inference using trained deep learning models
Generate cancer risk score (Low / Medium / High)
Display AI results and heatmaps to doctors
Store reports and images securely
Role-based access for patients, health workers, and doctors

Non-Functional Requirements

High accuracy and low latency for AI inference
Secure handling of medical data (encryption, access control)
Scalable cloud-based architecture
User-friendly interface for non-technical users
High availability and reliability

Technologies Requirements
Frontend: React.js, Camera APIs
Backend: Python (FastAPI / Flask) , REST APIs
Image Processing: OpenCV, NumPy, PIL
AI / Machine Learning: TensorFlow / PyTorch, CNN Models (ResNet / EfficientNet)
Decision & Analytics: Risk Scoring Engine,Scikit-learn, Pandas
Data & Security: Cloud Platform (AWS),Encrypted Storage, MongoDB, Role-Based Access Control
